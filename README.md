# Shahar Math

Hebrew, kid-friendly 80s-themed math site for 4th grade.

## Live Site
- GitHub Pages: https://niryariv.github.io/shahar_math/

## Features
- Short explanations in Hebrew
- Colorful 80s arcade vibe
- Multiplication, addition, subtraction, division
- Randomized exercises
- Positive feedback with sound + visual celebrations

## Local Run
Open `index.html` in a browser.
