# Changelog

All notable changes to this project will be documented in this file.

## [0.3.0] - 2026-02-10
### Added
- Sections for addition, subtraction, and division with themed explanations.
- Randomized exercises for all operations.
- Enhanced celebration effects with stickers and multiple sound variants.
- LTR equation layout (e.g., 3*4=?).

## [0.2.0] - 2026-02-10
### Added
- “More exercises” behavior that appends instead of replaces.
- Positive feedback emphasis with randomized messages.

## [0.1.0] - 2026-02-10
### Added
- Initial Hebrew 80s-themed multiplication lesson and exercises.
